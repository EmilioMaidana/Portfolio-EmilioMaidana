# Emilio Maidana — Portfolio

Portfolio profesional bilingüe (ES/EN) con dos perfiles intercambiables:
**Full-Stack Developer** y **Data Analyst**.

- Sitio 100 % estático: HTML + CSS + JS vanilla, sin build ni dependencias.
- Idioma detectado automáticamente según el navegador, con toggle manual ES/EN.
- Selector de perfil que adapta contenido, color de acento y CV descargable.

## Ver localmente

```bash
python -m http.server 4173
# → http://localhost:4173
```
